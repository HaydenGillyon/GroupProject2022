# Code Standards
- Python and Django backend
- JavaScript, React, HTML, CSS frontend
- Follow the corrections given by the linter (Super-Linter) upon pushing code in Python, JavaScript, or HTML

## Python
- Use 3.10.x
- Code to PEP-8

## Django
- Use 4.0.2

## React
- Use 17.0.2
